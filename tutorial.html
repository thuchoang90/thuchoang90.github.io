Computer Architecture Courses
Cources related to computer architecture: from studying the Verilog HDL coding, to studying the assembly coding, and finally, learn how to make a custom computer system and program C/C++ to it.

  • Verilog course: practices for new learners in studying Hardware Description Language (HDL).
  • Assembly course: exercises assembly coding, practice with the Altera's Nios-II processor.
  • System on Programmable Chip (SoPC) course: making a custom hardware computer system with the Altera's Nios-II and then programming C/C++ on it.


Matlab and Digital Signal Processing (DSP) Courses
Exercises for new learners using the Matlab tool, practice with DSP applications.
